Knowledge base: Graph systems

Wednesday, Jun 10, 2020| Tags: data, analytics, tech, graph

Graph Systems

Documenting a list of graph systems.

  1. Alibaba Graph Database - A real-time, reliable, cloud-native graph database service that supports property graph model.
  2. Amazon Neptune - Fully-managed graph database service.
  3. ArangoDB - OLTP Provider for ArangoDB.
  4. Bitsy - A small, fast, embeddable, durable in-memory graph database.
  5. Blazegraph - RDF graph database with OLTP support.
  6. CosmosDB - Microsoft’s distributed OLTP graph database.
  7. ChronoGraph - A versioned graph database.
  8. DSEGraph - DataStax graph database with OLTP and OLAP support.
  9. GRAKN.AI - Distributed OLTP/OLAP knowledge graph system.
  10. Hadoop (Spark) - OLAP graph processor using Spark.
  11. HGraphDB - OLTP graph database running on Apache HBase.
  12. Huawei Graph Engine Service - Fully-managed, distributed, at-scale graph query and analysis service that provides a visualized interactive analytics platform.
  13. IBM Graph - OLTP graph database as a service.
  14. JanusGraph - Distributed OLTP and OLAP graph database with BerkeleyDB, Apache Cassandra and Apache HBase support.
  15. JanusGraph (Amazon) - The Amazon DynamoDB Storage Backend for JanusGraph.
  16. Neo4j - OLTP graph database (embedded and high availability).
  17. neo4j-gremlin-bolt - OLTP graph database (using Bolt Protocol).
  18. OrientDB - OLTP graph database
  19. Apache S2Graph - OLTP graph database running on Apache HBase.
  20. Sqlg - OLTP implementation on SQL databases.
  21. Stardog - RDF graph database with OLTP and OLAP support.
  22. TinkerGraph - In-memory OLTP and OLAP reference implementation.
  23. Titan - Distributed OLTP and OLAP graph database with BerkeleyDB, Apache Cassandra and Apache HBase support.
  24. Titan (Amazon) - The Amazon DynamoDB storage backend for Titan.
  25. Titan (Tupl) - The Tupl storage backend for Titan.
  26. Unipop - OLTP Elasticsearch and JDBC backed graph.
